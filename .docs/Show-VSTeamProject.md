## DESCRIPTION

Opens the project in default browser.

You must call Set-VSTeamAccount before calling this function.

## EXAMPLES

### Example 1

```powershell
Show-VSTeamProject TestProject
```

This will open a browser to the TestProject site

## PARAMETERS
