Welcome to VSTeam. VSTeam is a [PowerShell module](https://www.powershellgallery.com/packages/VSTeam/) that wraps the [REST API provided by Team Foundation Server and Azure DevOps](https://cda.ms/ys). This allows you to access the power of Azure DevOps from [PowerShell on Windows, MacOS and Linux](https://github.com/PowerShell/PowerShell).
