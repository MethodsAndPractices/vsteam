## EXAMPLES

### Example 1: Get user profile by id

```powershell
$myProfile = Get-VSTeamUserProfile -Id f6e38832-6804-464f-9cf9-4aaa97327cfc
```

### Example 2: Get current users profile

```powershell
$myProfile = Get-VSTeamUserProfile -MyProfile
```

```yaml
Type: string
Parameter sets: Id
```
