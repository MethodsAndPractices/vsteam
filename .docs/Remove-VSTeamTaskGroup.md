### Example 1

```powershell
$projectName = "some_project_name"
$taskGroup = Get-VSTeamTaskGroup -Name "taskName" -ProjectName $projectName

$methodParameters = @{
   Id                       = $taskGroup.id
   ProjectName              = $projectName
   Force                    = $true
}

Remove-VSTeamTaskGroup @methodParameters
```
