## EXAMPLES

### Example 1

```powershell
Show-VSTeamPullRequest 3
```

This command will open a web browser with the pull request id of 3.

### Example 2

```powershell
Show-VSTeamPullRequest -Id 3
```

This command will open a web browser with the pull request id of 3.
